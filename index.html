Distribution platform for HTML5 Game
  • IGNSDK supports most HTML5 features
  • WEBGL support
  • Video and audio support
  • Flash  player plugin support
Distribution platform for CRUD Application
  • IGNSDK supports more DBMS (MySQL/MariaDB,SQlite)
  • HTML5 Web Database support
  • Local Storage API support
